# Weather Tracker

The Weather Tracker app allows the user to input a city and receive current weather data, as well as a 5 day forecast containing temp, conditional, wind and humidity information.

In order to use the app the user must type in a city in the search bar. The most relevant result will display. If the user would like to search more specifically, they can search for an exact location using the "City, Country" (for US cities, "City, State Abbreviation, US") format.
